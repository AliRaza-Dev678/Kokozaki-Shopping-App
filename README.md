# 🛒 Kokozaki - Social E-Commerce App 📱✨

**Kokozaki** is a professional, high-fidelity mobile application built with **Flutter** and **Dart**. It seamlessly blends modern e-commerce shopping with social interaction features, providing a complete user journey from product discovery to secure checkout and real-time engagement. 🚀

---

## 🌟 Key Features

### 🔐 **Secure Authentication**
* **Dynamic Login & Signup:** Smooth transitions with focus-state driven input validation.
* **Account Recovery:** Integrated Forgot Password and OTP verification flows.

### 🏪 **Shopping Experience**
* **Market Discovery:** Browse through an optimized list of local supermarkets. 🔍
* **Dynamic Product Views:** Interactive detail screens with image carousels and pricing logic.
* **Advanced Search:** Real-time search filtering for products and vendors.

### 🛒 **Cart & Checkout Lifecycle**
* **Smart Cart:** Manage order quantities with a modern, intuitive UI.
* **Multi-step Checkout:** Seamless flow from shipping address to payment selection. 🚚
* **Secure Payment Vault:** Custom-built UI for managing PayPal, MasterCard, Apple Pay, and more. 💳

### 💬 **Social & Engagement**
* **Real-time Style Chat:** Integrated messaging system for user-to-vendor communication.
* **Follower System:** Built-in community features with "Follow" and "Add Friends" logic.
* **Ratings & Reviews:** Comprehensive feedback system with interactive star ratings. ⭐️

### 👤 **Profile & Finance**
* **Account Settings:** Manage personal info, bank details, and security settings.
* **Order History:** Track previous purchases with detailed status reports. 📜
* **Earnings Dashboard:** Clean visualization of user balance and bank transfer options. 💰

---

## 🛠️ Tech Stack

* **Framework:** [Flutter](https://flutter.dev/) (v3.x) 🐦
* **Language:** [Dart](https://dart.dev/) 🎯
* **State Management:** `setState` (Optimized for performance)
* **Architecture:** Modular Component-Based Design
* **UI/UX:** Material Design 3 with custom-themed blue (`#647BFF`) and orange accents.

## 🚀 Getting Started

### Prerequisites
* Flutter SDK installed on your machine.
* Android Studio / VS Code.
* An Android/iOS Emulator or Physical Device.

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/kokozaki_app_project_1.git
   ```
2. **Navigate to the project folder:**
   ```bash
   cd kokozaki_app_project_1
   ```
3. **Install dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the application:**
   ```bash
   flutter run
   ```
